# ExpAffSVolCalib
Exponential Affine Stochastic Volatility Option Pricing Models Calibration - PhD project

The project was done for empirical testing in author's PhD research:

"Analytic estimation of parameters of stochastic volatility diffusion models with exponential-affine characteristic function for currency option pricing"

It is recommended to read the pdf file of this dissertation: https://arxiv.org/abs/2507.11868
